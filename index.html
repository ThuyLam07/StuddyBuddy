WebRTC: Passing SDP Manually Without Signaling

Instructions: Open two browser tabs side by side. Follow these steps to pass the SDP offer and answer. Each tab will represent a user:

  • User 1 (Tab 1)
  • User 2 (Tab 2)
Source Code

Step 1: User 1, click "Create Offer" to generate an SDP offer. Then, copy the offer from the text area below and share it with User 2.

Step 2: User 2, paste the SDP offer from User 1 into the text area above. Then click "Create Answer" to generate an SDP answer. Copy the answer and share it with User 1.

Step 3: User 1, paste the SDP answer from User 2 into the text area above and click "Add Answer" to complete the connection.
